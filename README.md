kdeplasma-applets-playbar
=========================

Client Mpris2, allows you to control your favorite media player, now supports Plasma 5

URL = https://github.com/audoban/PlayBar2

Install with kcp:

```
kcp -i kdeplasma-applets-playbar2
```
